Linux命令及双系统安装配置

Posted on 2018-11-01 | Visitors:

linux基本命令

Ubuntu16.04

  1. 跟工作目录相关命令
  • cd 返回根目录
  • cd + dirpath 更改工作目录至dirpath
  • cd .. 返回上一级目录
  • pwd 返回当前工作目录
  • ls 查看当前目录下的文件
  • ls -l 查看当前目录下的文件的详细信息
  • rm 删除文件
  • rmdir 删除文件夹
  • rm -rf 强制性删除非空目录
  • 复制文件或目录: cp -r 文件或目录名称 指定目录
  1. 开启与关闭真正终端
  • Ctrl + Alt + T 打开虚拟终端
  • ctrl + alt + f6 开启真正的终端
  • ctrl + alt + f7 关闭真正的终端
  1. 超级管理员
  • sudo 即super do 超级管理员
  • sudo -i 直接进入超级管理员模式
  • su + username 退出超级管理员模式

  1. nano+filename 在当前文件下新建文件，nano是ubuntu默认的编辑器

  2. 文件权限：r w x

  • r read 读的权限
  • w write 写的权限
  • x 执行的权限
  • r w x的值分别为4 2 1
  • rwx=7 r-w=6 r-x=5 w-x=3

  • 最前边的-代表文件，如果是d则代表是文件夹
  • 最前边的rw 表示该文件或文件所属的用户对该文件的操作权限
  • 中间的rw 表示该文件或文件所属的用户组对该文件的操作权限

  1. rm remove删除文件

  2. tail -n filename 查看文件内容

  3. man 查看帮助

  4. 从ubuntu终端进入Python和scrapy shell，再退回到终端的快捷键都是：Ctrl + D。

  5. 从终端中以窗口形式打开一个文件夹：nautilus dirname

  7. Ubuntu Unity 桌面快捷键切换窗口技巧：

  • 不同程序之间切换：alt + tab
  • 同一程序不同窗口之间切换：alt + `（tab键上方按键）
